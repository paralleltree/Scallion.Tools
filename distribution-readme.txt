
MikuMikuUtilsをダウンロード頂きありがとうございます。
本コマンド群は、MMDにおける利便性の向上を図って作成されたものです。
実行には.NET Framework 4.6が必要です。

○ 各コマンドの概要

【vmdscale】
モーションファイルを拡大・縮小して保存します。

【vmdcut】
モーションファイルの指定の範囲を切り出して出力します。

【vmdstat】
モーションファイルの情報を表示します。

○ 導入/使い方
圧縮ファイルを任意のディレクトリに解凍することで利用できます。
zipファイルがブロックされ、実行できない場合があるので、
zipファイルのプロパティにあるブロックの解除にチェックを入れてから解凍してください。
基本的にコマンドプロンプトから利用する形になります。
また、環境変数の設定でパスを通すことで、より簡単に実行することが可能です。

● パスの通し方 (Win10)
スタートボタンを右クリックして「システム」を選択します。
「システムの詳細設定」から「環境変数」をクリックします。
ユーザー環境変数の「Path」を選択した状態で、「編集」ボタンをクリックします。
「変数値」の最後に「;(解凍先パス)」を追記して「OK」をクリックします。
例: C:\foo;D:\MMD\MikuMikuUtils

● 使い方
スタートボタンの右クリックからコマンドプロンプトを起動します。
cdコマンドで任意のディレクトリに移動します。
例: cd D:\MMD
コマンドを実行します。パスを通していない場合はコマンドのフルパスを指定する必要があります。
このとき、--helpとともに実行することでヘルプが表示できます。
例: vmdscale --help
ファイルの上書き時には確認を求めますが、(y/N)に対してそのままエンターを押すとデフォルト値としてn(上書きしない)が選択されます。
